# cc-DnD-generator
Generate basic dungeon level and theme

## Description/plan

This project will consist of three stages:

1. Generating a layout for a small/medium/large dungeon based on user input
2. Select a theme for the dungeon based on news articles
3. Populate dungeon with appropraitely themed enemies

Potential additions:

1. Create characters to complete the dungeon with
2. Set a challenge level/rate for the dungeon
